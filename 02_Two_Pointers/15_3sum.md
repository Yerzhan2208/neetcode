## 🎯 Core Intuition
I was thinking of doing the same pattern as in the last problem, so I needed to sort the array and then make one index fixed, and left and right pointers dynamic relatively to the fixed one. Thus, if we found, we add this to the resulting array and continue searching by progressing left pointer by 1.


## 🚀 Optimal Solution

### Python Implementation
```python
class Solution:
    def threeSum(self, nums: List[int]) -> List[List[int]]:
        nums = sorted(nums)
        n = len(nums)
        res = []
        for m in range(n):
            l = m + 1
            r = n - 1
            while l < r:
                sm = nums[m] + nums[l] + nums[r]
                if sm > 0:
                    r -= 1
                elif sm < 0:
                    l += 1
                else:
                    if [nums[m], nums[l], nums[r]] not in res:
                        res.append([nums[m], nums[l], nums[r]])
                    l += 1    
        return res
